# Домашняя работа 24.1 "Вьюсеты и дженерики".

## Функционал.
### Создан django проект с моделями:
   - Пользователь:
        - все поля от обычного пользователя, но авторизацию заменить на email;
        - телефон;
        - город;
        - аватарка.
   - Курс:
        - название,
        - превью (картинка),
        - описание.
   - Урок:
        - название,
        - описание,
        - превью (картинка),
        - ссылка на видео.

### Описан CRUD:
    
   - Для модели "Курс" через ViewSets
   - Для моделей "Урок" и "Пользователь" через Generics-классы

### Требования к установке.
- В PostgreSQL должна быть создана DB - products

Есть возможность добавить учетную запись администратора
```
> python manage.py ucreate
```
